# 🤖 AI Education & Interaction Telegram Bot Assistant

Універсальний Telegram-бот на базі **GPT-4**, створений для навчання, отримання миттєвих відповідей та інтерактивного спілкування. Бот має модульну структуру, підтримує декілька мов та зберігає контекст розмови.

## 🌟 Основні режими роботи

### 🎓 Освіта та перевірка знань
* **Quiz (Квіз)**: Генерація питань з програмування, математики та біології.
    * **Anti-repeat**: Бот пам'ятає останні питання, щоб не дублювати їх.
    * **Smart Scoring**: Точний підрахунок балів.
* **Vocabulary**: Тренажер для вивчення англійських слів із перевіркою через AI.

### 💬 Інтерактив та AI-помічник
* **Tutor (Talk)**: Повноцінний діалог із ШІ, який пам'ятає попередні репліки (Sliding Window Context).
* **AI Assistant**: Режим швидкої відповіді на будь-яке запитання без збереження історії.
* **Random Fact**: Отримання випадкового цікавого факту від нейромережі.

## ⌨️ Командний інтерфейс

| Команда | Функція |
| :--- | :--- |
| `/start` | Запуск бота та головне меню |
| `/random` | Випадковий цікавий факт  |
| `/gpt` | Пряме запитання до ChatGPT |
| `/talk` | Спілкування з AI-особистістю |
| `/quiz` | Запуск квізу за обраною темою |
| `/vocabulary` | Тренажер іноземних слів |

## 🛠 Технічні особливості
* **Керування пам'яттю**: Система автоматично очищує історію повідомлень, залишаючи системний промпт та останні 10-20 реплік (задається константою) для економії токенів.
* **Локалізація**: Всі тексти та системні повідомлення винесені в `JSON` файли (`uk.json`, `en.json`), що дозволяє легко додавати нові мови.
* **State Machine**: Використання `ConversationHandler` для стабільної роботи кнопок та навігації.

## 🚀 Швидкий старт

1. **Налаштуйте середовище**:
   Створіть файл `.env` і додайте:
   ```env
   TELEGRAM_TOKEN=your_telegram_bot_token
   OPENAI_API_KEY=your_openai_api_key
   ```
2. **Встановіть залежності:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустіть бота:**
   ```bash
   python main.py
   ```
